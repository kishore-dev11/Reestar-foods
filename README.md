# Reestar Restaurant Website

A simple restaurant website built using **HTML and CSS**. This project displays a food menu with images, an order button, and a contact form. It is a beginner-friendly front-end project to practice basic web design and layout.

## Features

* Simple and clean restaurant menu layout
* Food cards with images and descriptions
* Order button design
* Contact form section
* Built using pure HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to practice basic front-end development skills such as layout design, styling, and structuring a simple website.
